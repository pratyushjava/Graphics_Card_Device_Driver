CPSC 8220 Project 1 - Device Driver


The source code contains 3 files: 

    kyouko3.h - contains the header files common to user code and kernel code
    mymod.c - kernel code
    spherenew.c - user code

It also includes a shell file “reinstall.sh” 

To run this project:

    make
    ./reinstall.sh
    ./a.out


To run FIFO, Press 0 and to run DMA, Press 1.

The folder also contains the screenshots of the output.
